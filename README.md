## 概要
Gemini Pro APIを使ってテキスト生成を行います。

例) この画像に何が写っているか教えてください。

## インストール

```
npm i node-red-contrib-simple-gemini
```

or

AdminタブからInstall

## 使い方
### 入力項目

|項目|説明|
|--|--|
|payload|生成したいプロンプトを入力します。|
|API Key|Gemini ProのAPIキーを設定します。|

### 出力項目

|項目|説明|
|--|--|
|payload|生成されたテキストが返ってきます。|
